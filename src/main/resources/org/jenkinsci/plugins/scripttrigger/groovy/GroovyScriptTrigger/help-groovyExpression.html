Give a groovy script.
If the script result is evaluate to true, a build is scheduled.

A script example:
File f = new File('/Users/jenkinsci/deploy/artifact.ext')
f.exists()

Scheduler field supports parameters in format:
#@param <name> [value]

Parameters is applied to a cron item above which they are listed.
Parameters override default job's parameters values.
No quotes are needed in case a parameter value contains whitespaces.
Parameter's value may be omitted. In this case it will be assumed as empty value.

Predefined variables:

parameters
Parameters defined in Scheduler.
